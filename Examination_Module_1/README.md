**Задача:**

1. Создать проект, используя шаблон Console Application;
2. Создать программу, которая подсчитывает общую стоимость всех введенных продуктов и выводит результат на экран;
 
**Дополнительная задача:**

1. Реализовать механизм подсчета общей стоимости продуктов с учетом скидки;

**Исходные данные:**

1. Список товаров (около 5-и) и их стоимость за единицу;

**Приблизительный алгоритм решения**

1. Вывести на экран список продуктов, их стоимость и порядковый номер для дальнейшего выбора;
2. В бесконечном цикле с использованием условной (-ых) конструкции(-й) попросить пользователя ввести номер товара и его количество. При этом заполняются данные в строковую переменную и считается общая сумма;
3. Предусмотреть условие выхода из бесконечного цикла;
4. Дополнительно: спросить у пользователя наличие скидки, подсчитать итоговую сумму с учетом скидки и заполнить данные в строковую переменную.
5. Вывести строковую переменную на экран, в которой будет хранится информация о наименованиях выбранных продуктов, их количества и итоговая сумма

Рекомендуемые конструкции:

1. if;
2. switch-case;
3. while;
4. continue, break;
5. int.Parse
